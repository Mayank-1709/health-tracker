# Personal Health Tracker

A web application built with Python and Flask to help manage and visualise personal health metrics.

## Features
- Daily weight logging
- Medication tracking
- Personal notes
- Data persistence with SQLite database

## Built With
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML/CSS

## Background
Built as a personal project after a cardiac health event in 2025. 
Designed to track recovery progress including weight, medication adherence, and daily notes.

## How to Run
1. Clone the repository
2. Create a conda environment: `conda create -n health_tracker python=3.11`
3. Activate it: `conda activate health_tracker`
4. Install dependencies: `pip install flask flask-sqlalchemy`
5. Run: `python app.py`
6. Open browser at `http://127.0.0.1:5000`
